# Student Management System

## Tech Stack
- Spring Boot
- Spring Security
- JPA
- Hibernate
- MySQL

## Features
- Admin Authentication
- Student Admission
- Course Management
- Course Assignment
- Student Profile Update
- Search Students by Name
- Search Students by Course

## Setup Instructions
1. Clone repo
2. Configure MySQL in application.properties
3. Run:
   mvn clean install
4. Start application:
   mvn spring-boot:run
